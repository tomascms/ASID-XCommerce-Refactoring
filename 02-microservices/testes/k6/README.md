# k6 Test Scripts (consolidated)

Esta pasta contém os scripts k6 consolidados originalmente em `testScenarios/`.

Nota: os scripts podem apontar para o Gateway em `http://localhost:9000`. Se algum script
apontar para `8080` ou usar login local, atualiza o script antes de correr.

Para correr todos os testes:

```bash
./run-all.sh
```
